# 韓国語ー日本語文字・発音対応票

このレポジトリは韓国語文字（ハングル）を読みたい人達をために作りました。もし違い部分を見つかったら、どうかPRお願いいたします。

## なぜこんあレポジトリを作ったんですか？

ただ韓国のかたも、日本のかたも、皆さんが共に双方の文字を読めればいいだと思いました。僕自分の日本語は下手くそですが、もっと頑張って誰でも理解安いな説明を書きます。

## 1. 子音と母音

ハングルの子音は「ㄱ,ㄴ,ㄷ,ㄹ,ㅁ,ㅂ,ㅅ,ㅇ,ㅈ,ㅊ,ㅋ,ㅌ,ㅍ,ㅎ」の１４字があり、

母音は「ㅏ,ㅑ,ㅓ,ㅕ,ㅜ,ㅠ,ㅡ,ㅣ」１０字があります。

この文字を調合して一の音節をつくります。
