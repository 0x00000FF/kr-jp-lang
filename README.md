# 韓国語ー日本語文字・発音対応票

このレポジトリは韓国語文字（ハングル）を読めたい人達をために作りました。もし違い部分を見つかったら、どうかPRお願いいたします。

## なぜこんなレポジトリを作ったんですか？

ただ韓国のかたも、日本のかたも、皆さんが共に双方の文字を読めればいいだと思いました。僕自分の日本語は下手くそですが、もっと頑張って誰でも理解安いな説明を書きます。
